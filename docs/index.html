자바스크립트 영상 재생기 v1.0

[주요 기능]

1. 다양한 기능 지원: 재생/정지/되감기/빨리감기/특정위치로 이동/배속변경/반복여부/풀스크린 등
   ※ 재생 시점을 숫자 및 게이지로 표시하며, 게이지 클릭 시 클릭한 해당 시점으로 이동 재생
2. 크로스브라우징 테스트 완료 : IE(11~) / Firefox / Chrome / Edge / Whale / Opera 등 주요 브라우저 모두 지원
   ※ 모바일 T/O 필요
3. 모바일 환경에서 터치/더블탭을 통한 조작 지원
   ※ 모바일 T/O 필요
4. 키보드를 통한 컨트롤러 조작 지원
5. 기능 작동 시 피드백 애니메이션 구현 (해당 기능 버튼이 화면에 표시되는 등)
6. 객체가 패키지 형태로 모듈화되어 다양한 환경에 손쉽게 적용 가능

  vid.calc.getNewCurrentTimeByMousePos
  vid.calc.getCurrHoveredPerc
- 볼륨 게이지, 재생 게이지 드래그 가능하게 바꾸기

- playsinline 문제 https://stackoverflow.com/questions/19521667/disable-fullscreen-iphone-video
- iOS, MacOS에서 무한로딩 현상 발생 해결, 그외 모바일에서 구현 가능한지 테스트해 보기 (iOS / Android)
- 미디어쿼리 똑바로 적용
- 브라우저별 지원 가능한 동영상/코덱 테스트, 정리 필요함
